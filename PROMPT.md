# Assignment Prompt Template

This file provides a template for providing assignments.

## Prompt 1: The Assignment

**Topic:** [Your Topic Here]

**Learning Resources:**
- [Resource 1]
- [Resource 2]
- ...

**Assignment:**
[Detailed description of the assignment]

**Instructions:**
- Create a plan to implement this assignment.
- Create a `docs/` directory with documentation for the topics covered.

## Prompt 2: Saving the Plan

Before starting the implementation, write the plan into a `DAY-X.md` file (where X is the day number).

## Prompt 3: Documenting the Implementation

After completing the implementation, document all the changes made in a `DAY-X-Steps.md` file.

## Prompt 4: Documenting the Tests

Create a `DAY-X-Test.md` file with `curl` commands (or other testing instructions) to test the implemented features.
